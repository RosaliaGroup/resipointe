# Iron Pointe Newark — Luxury Living

**Multi-campaign marketing website** for Iron Pointe Newark luxury apartments at 39 Madison St, Newark, NJ.

## Features

- 8 targeted audience campaigns (NYC Commuters, Flight Attendants, Pilots, Relocation, Pet Lovers, etc.)
- Interactive cost calculator with parking & pet add-ons
- Floor plan browser (1 BR & 2 BR)
- Rooftop & gym gallery
- Neighborhood guide
- SEO-optimized with Open Graph and Twitter Card meta tags
- Responsive design
