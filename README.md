# Resipointe — Newark Luxury Living

A full-stack real estate marketing and management platform for Resipointe's Newark, NJ multi-property portfolio.

**Live Site:** https://www.resipointe.com  
**Tech Stack:** React 19 · TypeScript · Tailwind CSS 4 · tRPC · Express · MySQL · Drizzle ORM

## Features

- **5-Property Portfolio** — Iron Pointe, University Pointe, River Pointe, 502 Market St, 556 Market St with real photos
- **6 Campaign Landing Pages** — NYC Commuters, No-NY-Tax, Aviators/Pilots, Gen Z, Modern Finishes, Amenities
- **AI Chatbot** — LLM-powered assistant for property Q&A, unit comparison, and tour booking
- **Appointment Booking** — Calendar-based tour scheduling with lead capture and owner notifications
- **Collaboration Hub** — Tenant/developer surveys, feature voting, and insights dashboard
- **Social Media Generator** — AI-powered captions, hashtags, and CTAs for Instagram, Facebook, LinkedIn
- **Floor Plan Browser** — Filter by bedrooms, square footage, availability with pricing
- **Cost Calculator** — NYC vs Newark comparison with 2-months-free effective pricing
- **Property Gallery** — Lightbox gallery categorized by exterior, interior, amenities, neighborhood
- **Full SEO** — JSON-LD structured data, Open Graph, Twitter Cards, XML sitemap, robots.txt
